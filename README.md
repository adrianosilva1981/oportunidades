# Oportunidades

## Instruções

### Na raiz do repositório:

* `docker-compose up -d`

### No prompt de comando, navegar até `api` e executar:

* `composer install`
* `php artisan migrate`
* Subindo o serviço: `php artisan serve`

### Na pasta `app` executar:

* `npm i`
* Subindo o serviço: `npm start`
